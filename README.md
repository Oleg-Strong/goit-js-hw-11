# Gallery

## Description

The "Gallery" project is a small web application for searching pictures by
keyword, and conveniently viewing them. It is built using Java Script, using
libraries. The "modern-normalize" library normalizes styles. The "Axios" library
is used to interact with the API. The "notiflix" and "simplelightbox" libraries
are used to to improve the user experience.

## Installation

1. Clone the repository: `https://github.com/Oleg-Strong/goit-js-hw-11`
2. Open the project in the code editor
3. In the terminal, install the dependencies with the command: `npm install`

## Usage

1. Run the application: `npm start`
2. Open your web browser and visit: `http://localhost:1234`
